OS1 Assignment 2

This program tries to find all the perfect numbers less than or equal to 'n' using multithreading. The program also uses an integer 'k', the number of threads to create.
Input is given using command line argument. The program takes the name of file containing value of n and k as input.

Execution:
(i) Compile the c file using following command:
gcc -pthread Assgn2Src-CO21BTECH11002.c -o Assgn2Src-CO21BTECH110002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./Assgn2Src-CO21BTECH11002.out in.txt
