This program tries to find all the perfect numbers less than or equal to 'n' using multiprocessing. The program also uses an integer 'k', the number of processes to create.
Input is given using command line argument. The program takes the name of file containing value of n and k as input.

Compile the c file. Say the executable is a.out. Run the executable file with the name of input file as the command line argument.
