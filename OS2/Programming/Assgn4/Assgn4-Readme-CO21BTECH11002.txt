OS2 Assignment 4

Input is given using command line argument. The program takes the name of file containing value of n (no. of cars), p (no. of passengers), lam_c (parameter for exponential waiting time of car), lam_p (parameter for exponential waiting time of passengers) and k (no. of ride requests of each passenger) as input.

Executing the file:

(i) Compile the cpp file using following command:
g++ Assgn4-Src-CO21BTECH11002.cpp -o Assgn4-Src-CO21BTECH11002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./Assgn4-Src-CO21BTECH11002.out inp-params.txt
