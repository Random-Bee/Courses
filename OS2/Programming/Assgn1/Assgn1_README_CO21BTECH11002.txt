OS1 Assignment 2

This program calculates a approximate value of pi using Monte-Carlo method using multithreading.
Input is given using command line argument. The program takes the name of file containing value of n (no. of points to generate and check) and k (no. of threads to use) as input.

Execution:
(i) Compile the cpp file using following command:
g++ -pthread Assgn1_Src_CO21BTECH11002.cpp -o Assgn1_Src_CO21BTECH11002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./Assgn1_Src_CO21BTECH11002.out inp.txt
