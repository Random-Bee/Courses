Distributed Computing Programming Assignment 2

Input File (inp-params.txt):
n, k, a, b

n - number of processes
k - number of CS entries
a, b - exponential distribution parameters

Executing the file:

Make sure that the input file (inp-params.txt) is in the same directory as the executable file.

RC:

(i) Compile the cpp file using following command:
g++ RC-CO21BTECH11002.cpp -lzmq -o RC

(ii) Run the executable file using following command:
./RC

SK:

(i) Compile the cpp file using following command:
g++ MK-CO21BTECH11002.cpp -lzmq -o MK

(ii) Run the executable file using following command:
./MK
