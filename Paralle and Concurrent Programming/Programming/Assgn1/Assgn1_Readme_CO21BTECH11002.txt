Parallel and Concurrent Programming Assignment 1

This program finds all the prime numbers less than a given number N.
Input is given using command line argument. The program takes the name of file containing value of n (number of digits) and m (no. of threads to use) as input.

Executing the file:
(i) Compile the cpp file using following command:
g++ Src-CO21BTECH11002.cpp -o Src-CO21BTECH11002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./Src-CO21BTECH11002.out inp-params.txt
