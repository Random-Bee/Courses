Parallel and Concurrent Programming Assignment 4

Input is given using command line argument. The program takes the name of file containing value of nW (number of writer threads), nS (number of snapshot threads), M (capacity of shared array), μw, μs, and k (number of snapshots by each thread) as input.

Executing the file:

OFS:

(i) Compile the cpp file using following command:
g++ obs-CO21BTECH11002.cpp -o obs-CO21BTECH11002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./obs-CO21BTECH11002.out inp-params.txt

WFS:

(i) Compile the cpp file using following command:
g++ wfs-CO21BTECH11002.cpp -o wfs-CO21BTECH11002.out

(ii) Run the executable file with the name of input file as the command line argument using following command:
./wfs-CO21BTECH11002.out inp-params.txt
